Kensa
===========
Scan and generate QR codes through SMS/MMS messages.

### Setup
Install server dependencies.
```
$ [sudo] npm install
```
Add ```.env``` file with the following variables:
```
TWILIO_ACCOUNT_SID=XXXXXXXXXXXXXXXXXXX
TWILIO_AUTH_TOKEN=XXXXXXXXXXXXXXXXXXX
```
Start the server.
```
$ node app.js
```
